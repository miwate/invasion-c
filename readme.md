# INVASION !!
Projet de C

## Dépendances
Installation des dépendances

```bash
sudo apt-get install libsdl2-dev
sudo apt-get install libsdl2-image-dev
```

## Compilation
### Compilation du projet

```bash
make
```
- Exemple : lancer le niveau expert
```bash
make expert
```
### Nettoyage
- Standard
```bash
make clean
```
- Complet
```bash
make cleanAll
```

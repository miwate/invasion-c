# INVASION !!

## Dépendances
``sudo apt-get install libsdl2-dev
sudo apt-get install libsdl2-image-dev``

## Compilation
Compiler les fichiers sources
``make``
Démarrer une partie
``make foobar``
Exemple : démarrer le niveau expert
``make expert``
Effacer les fichiers compilés
``make clean``
Effaçage complet des fichiers comilés
``make cleanAll``

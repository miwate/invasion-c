Ce fichier permet de compiler, exécuter et gérer les fichiers du projet. 
Pour compiler et accéder au jeu, suivez les indications de ce fichier texte.

Afin de lancer le niveau correspondant à celui voulu, il faut exécuter dans le terminal (en étant dans le bon répertoire au préalable) : make niveau
où niveau correspond au niveau choisi, les niveaux disponibles étant : facile, ...

Puis l'utilisation dans le terminal de : make cleanAll
permet de fermer le jeu exécuté par le programme. Vous pourrez alors recommencer avec un niveau différent ou le même (ou arrêter de jouer) !
